Reflectie op de Opdracht

Deze site was gemaakt als opdracht voor informatica. De basis voor de site was gemaakt door chatgpt, daarna heb ik zelf op w3schools de code gevonden om de site groter te maken. Opzich was het handig om voor het eerste stukje van de site chatgpt te gebruiken, je krijgt dan een heel stuk code in een paar seconden en dan is het even snel copy-pasten. Daarna daarentegen kan chatgpt wel helpen met code voor speciale dingetjes, maar dan moet je wel zelf nog de code veranderen zodat het goed in de site past.
